# Keil Install

Keil　μVersionは、プロジェクト管理、実行環境、ソースコード編集、プログラムデバックをすべて揃ったARMの純正コンパイラ搭載の統合開発環境IDEです。

有償版とコード容量制限付きの体験版(32KB)があります。

※Keil有償版はサポートが受けられます。開発時間の短縮が必要なときや開発費に余裕がある場合はこちらがお勧めです。
